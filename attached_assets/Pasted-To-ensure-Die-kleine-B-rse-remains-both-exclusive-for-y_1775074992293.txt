### 2. Static Category View (SEO Optimization)
For **Die kleine Börse**, we want Google to see all categories immediately. Even if a category like "Real Estate" is currently empty, the link and the keyword must exist on the homepage.

```tsx
const ALL_CATEGORIES = [
  { id: 'living-interior', label: 'Living & Interior', icon: '🪑' },
  { id: 'fashion-accessories', label: 'Fashion & Accessories', icon: '👜' },
  { id: 'art-collectibles', label: 'Art & Collectibles', icon: '🎨' },
  { id: 'tech-electronics', label: 'Tech & Gadgets', icon: '⌚' },
  { id: 'vehicles-mobility', label: 'Vehicles & Mobility', icon: '🏎️' },
  { id: 'real-estate', label: 'Real Estate', icon: '🏢' },
  { id: 'leisure-hobbies', label: 'Leisure & Hobbies', icon: '🎾' },
  { id: 'services', label: 'Services', icon: '🤝' }
];

export function CategoryGrid() {
  return (
    <section className="py-12">
      <h2 className="text-xs uppercase tracking-[0.2em] text-slate-400 mb-8 font-bold text-center">
        Explore Collections
      </h2>
      <div className="grid grid-cols-2 md:grid-cols-4 gap-4">
        {ALL_CATEGORIES.map((cat) => (
          <a 
            href={`/category/${cat.id}`} 
            key={cat.id}
            className="group p-8 border border-slate-100 rounded-3xl text-center hover:border-slate-300 hover:shadow-sm transition-all duration-300"
          >
            <div className="text-3xl mb-4 grayscale group-hover:grayscale-0 transition-all">
              {cat.icon}
            </div>
            <span className="text-sm font-medium text-slate-900">{cat.label}</span>
          </a>
        ))}
      </div>
    </section>
  );
}
```

### **Strategic SEO & Security Benefits**

1.  **Invisible Administration:** Normal users will never even know the Admin Dashboard exists. This reduces the risk of hacking attempts on your internal tools.
2.  **Internal Linking (SEO):** By displaying all categories on the homepage (even empty ones), you create a permanent internal link structure. This helps Google index your category pages faster so they are ready the moment the first car or house is listed.
3.  **Trust & Growth:** A full category grid makes "Die kleine Börse" look established and professional from day one, rather than looking like an "empty" new site.
